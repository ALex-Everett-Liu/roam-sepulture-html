设置 Git Bash 环境变量

Claude Code 配置指南
首页 编程 设置 Git Bash 环境变量

问题描述

启动 Claude Code 时遇到以下错误提示：

Launching Claude Code... Claude Code on Windows requires git-bash (https://git-scm.com/downloads/win). If installed but not in PATH, set environment variable pointing to your bash.exe, similar to: CLAUDE_CODE_GIT_BASH_PATH=C:\Program Files\Git\bin\bash.exe

Git Bash 路径确认

首先确认你的 Git Bash 安装位置：

E:\
scoop\
apps\
git\
2.50.0\
bin\
bash.exe
目标路径：
E:\scoop\apps\git\2.50.0\bin\bash.exe

解决方案

设置环境变量 CLAUDE_CODE_GIT_BASH_PATH 指向你的 Git Bash 路径。

变量名：
CLAUDE_CODE_GIT_BASH_PATH

变量值：
E:\scoop\apps\git\2.50.0\bin\bash.exe

方法 1：通过 PowerShell 设置（推荐）

1
PowerShell 管理员运行
以管理员身份打开 PowerShell

按 Win + X → 选择 "Windows PowerShell（管理员）"

执行设置命令
[Environment]::SetEnvironmentVariable("CLAUDE_CODE_GIT_BASH_PATH", "E:\scoop\apps\git\2.50.0\bin\bash.exe", "Machine")
提示： 使用 "Machine" 表示系统级环境变量（需要管理员权限）
Tip: Use "Machine" for system-wide environment variables (requires admin privileges)
若仅当前用户使用可替换为 "User"
Note: Replace with "User" for current user only

方法 2：手动设置环境变量

2
图形界面设置
打开系统设置
  • 按 Win + R → 输入 sysdm.cpl → 回车
  • 或：控制面板 → 系统和安全 → 系统 → 高级系统设置
设置环境变量
  • 点击"环境变量" → 在"系统变量"下点击"新建"
  • 变量名：CLAUDE_CODE_GIT_BASH_PATH
  • 变量值：E:\scoop\apps\git\2.50.0\bin\bash.exe
  • 点击所有"确定"保存

验证是否生效

打开新的 PowerShell 或 CMD

重要！必须新开窗口，否则不会识别新设置的环境变量
Important! Must open a new window, otherwise the newly set environment variable won't be recognized

执行验证命令
# PowerShell 中执行 $env:CLAUDE_CODE_GIT_BASH_PATH # CMD 中执行 echo %CLAUDE_CODE_GIT_BASH_PATH%
预期输出

应显示你的路径：

E:\scoop\apps\git\2.50.0\bin\bash.exe

注意事项

路径有效性：
  • 确认文件真实存在：在文件资源管理器地址栏输入路径验证
  • 若 Scoop 更新 Git 导致路径变化（如版本号改变），需同步更新此变量
权限问题：
  • 方法1需要管理员权限，方法2普通用户可操作（选"用户变量"）
重启应用：
  • 设置后需重启 Claude Code 应用才能识别新变量
💡 专业提示： 由于你通过 Scoop 安装 Git，路径中的版本号 2.50.0 可能在更新后变化，建议定期检查路径有效性。如需永久解决方案，可考虑在 Scoop 中创建 Git 的持久符号链接。
💡 Pro Tip: Since you installed Git via Scoop, the version number 2.50.0 in the path may change after updates. It's recommended to regularly check path validity. For a permanent solution, consider creating a persistent symbolic link for Git in Scoop.
