Gallery Management

How to manage your gallery:

1. Edit JSON file: Modify images_data_groups.json to add/update images

2. Image Groups: Create collections with major images and subsidiary images

3. Import/Export: Use the buttons above to manage your gallery data

4. Structure: Each image needs: title, description, src (for major images), and display properties

JSON Format Example 
{
  "images": [
    {
      "id": 1,
      "title": "Sunset Beach Collection",
      "description": "Beautiful sunset at the beach with multiple perspectives",
      "src": "path/to/your/image.jpg",
      "ranking": 9.2,
      "tags": ["sunset", "beach", "ocean"],
      "width": "600px",
      "height": "400px",
      "gridSpan": 2,
      "isMajor": true,
      "groupId": "group-1"
    },
    {
      "id": 2,
      "title": "Sunset Beach - Wide Shot",
      "src": "path/to/subsidiary1.jpg",
      "width": "400px",
      "height": "150px",
      "gridSpan": 2,
      "isMajor": false,
      "groupId": "group-1",
      "majorImageId": 1
    }
  ]
}

Major Images: Include ranking, tags, description (used for filtering/sorting)
Subsidiary Images: Inherit metadata from major image, only need src, width, height, gridSpan
Standalone Images: Set isMajor: true without groupId for individual display
