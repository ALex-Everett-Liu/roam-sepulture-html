Section Title: Python Lists: Ordered Collections

Python list indexing visualization

Creating and Manipulating Lists

Python lists are ordered, mutable collections that can store items of different types. They support fast O(1) indexing and provide numerous built-in methods for manipulation.

list_operations.py 
# Creating and manipulating lists
fruits = ['apple', 'banana', 'orange']
fruits.append('grape')        # Add to end
fruits.insert(1, 'kiwi')      # Insert at index
first_item = fruits[0]        # Access by index
fruits.remove('banana')       # Remove by value
popped = fruits.pop()         # Remove and return last
print(f"List: {fruits}")
print(f"Length: {len(fruits)}")

Performance Tip: List comprehensions are generally faster than equivalent for loops for creating new lists. This has been extensively studied in [1] and verified through practical benchmarks [2].
